# AINotesHub

A .NET 8 solution combining a WPF Desktop App and a REST API with shared models.

## 🧩 Structure
AINotesHub/
├── AINotesHub.Shared/ → Shared models & helpers
├── AINotesHub.API/ → ASP.NET Core REST API
└── AINotesHub.WPF/ → WPF Desktop Client

## ⚙️ How to Run
1. Clone the repo  
   `git clone https://github.com/<your-username>/AINotesHub.git`
2. Open `AINotesHub.sln` in Visual Studio
3. Set **Multiple Startup Projects** → Start both **API** & **WPF**
4. Run the solution 🎯

## 🧰 Tech Stack
- .NET 8 / C#
- ASP.NET Core Web API
- WPF (MVVM)
- Entity Framework Core
- Clean Architecture
